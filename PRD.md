# BioQuest 产品需求文档（PRD）

## 1. 产品概述

BioQuest 是一个面向中学生物学习的智能化在线平台。本 PRD 在现有题库、练习、社区、信用（CR）系统的基础上，新增四大核心模块：

1. **智能错题管理**：拍照/输入错题后，AI 自动识别知识点、定位教材章节、分析错误原因、生成关联知识图谱，并推送相关练习题。
2. **Canvas 生物过程可视化**：基于 HTML5 Canvas 的交互式动画，覆盖有丝分裂、减数分裂、DNA 复制等核心生物过程，支持拖拽、缩放、步进播放与知识点标注。
3. **虚拟生物实验室**：网页端交互式模拟实验，覆盖显微镜观察、色素提取、DNA 粗提取、微生物培养等高频考点，AI 引导操作步骤并实时反馈。
4. **学习管理工具集成**：整合课程表、待办清单、番茄钟、笔记、倒计时、计算器、翻译工具等，形成连续的学习节奏，解决考前"不知道先做什么"的痛点。

## 2. 目标用户

- **核心用户**：高中生物学学生（人教版必修 + 选择性必修）。
- **次要用户**：生物竞赛生、教师、自学生。
- **使用场景**：课后作业订正、考试复习、日常学习管理、考前冲刺规划。

## 3. 功能模块详细需求

### 3.1 智能错题管理

#### 3.1.1 错题录入
- **拍照录入**：调用设备摄像头或上传图片，AI OCR 提取题干、选项、学生作答与正确答案。
- **手动录入**：支持文本输入、LaTeX/公式编辑、图片附件。
- **从练习导入**：练习页面一键将错题加入错题本。

#### 3.1.2 AI 分析
- **知识点识别**：自动标注题目涉及的核心概念（如"光合作用光反应""减数分裂同源染色体分离"）。
- **教材章节定位**：映射到人教版高中生物章节（必修1第5章第4节等）。
- **错误原因分析**：区分"概念理解错误""审题错误""计算错误""记忆遗忘""混淆相似概念"等类型。
- **关联知识图谱**：基于题目知识点，从平台知识图谱中抽取前驱、后继、相关概念，形成可视化子图。

#### 3.1.3 复习与推送
- **间隔重复复习**：集成 FSRS 算法，根据掌握程度动态安排复习时间。
- **关联练习推送**：针对薄弱知识点，从题库中筛选 3-5 道相似题或变式题。
- **错题统计**：按知识点、错误原因、时间维度生成错题报告。

#### 3.1.4 数据模型
```
wrong_questions
- id (uuid)
- user_id (uuid)
- question_text (text)
- user_answer (text)
- correct_answer (text)
- subject (text)
- concept (text)
- textbook_chapter (text)
- error_reason (text)
- knowledge_graph_nodes (text[])
- review_due (timestamptz)
- fsrs_state (jsonb)
- created_at (timestamptz)
```

### 3.2 Canvas 生物过程可视化

#### 3.2.1 过程库
首批支持以下过程的交互式动画：
- 有丝分裂（前期、中期、后期、末期）
- 减数分裂 I / II
- DNA 复制（半保留复制、冈崎片段）
- 转录与翻译
- 光合作用（光反应、暗反应）
- 细胞呼吸（糖酵解、 Krebs 循环、电子传递链）
- 神经冲动传导

#### 3.2.2 交互能力
- **步进播放**：支持"上一步/下一步/自动播放/暂停/重置"。
- **拖拽与缩放**：鼠标/触屏拖拽画布，滚轮/捏合缩放。
- **热点标注**：点击细胞器或分子弹出知识点卡片。
- **速度控制**：0.25x - 2x 播放速度。
- **对比模式**：并排放置两个过程（如有丝分裂 vs 减数分裂）。

#### 3.2.3 AI 生成扩展
- 教师或学生输入新的生物过程描述，AI 生成该过程的步骤脚本与关键帧定义，经人工确认后渲染为动画。

#### 3.2.4 技术要点
- 使用原生 HTML5 Canvas 2D 渲染，保证移动端兼容性。
- 动画状态机管理当前步骤、过渡进度、用户交互状态。
- 矢量元素（染色体、细胞、分子）使用 Canvas Path 绘制，支持高清屏缩放。

### 3.3 虚拟生物实验室

#### 3.3.1 实验库
首批覆盖高频考点实验：
- 显微镜观察（植物细胞、动物细胞、有丝分裂装片）
- 叶绿体色素提取与分离（纸层析法）
- DNA 粗提取与鉴定
- 微生物培养与计数
- 酶活性影响因素探究
- 质壁分离与复原

#### 3.3.2 交互流程
- **实验准备**：选择实验，AI 介绍目的、原理、材料清单。
- **步骤引导**：左侧显示当前步骤，右侧为虚拟实验台。
- **拖拽操作**：从试剂架/工具栏拖拽器材到实验台正确位置。
- **实时反馈**：操作正确进入下一步；操作错误弹出提示、说明原因并允许重试。
- **数据记录**：自动记录实验现象与数据，生成实验报告。

#### 3.3.3 AI 辅导
- **智能提示**：当学生连续 2 次操作错误时，AI 给出具体引导。
- **原理解说**：学生可随时点击"为什么"查看该步骤的生物学原理。
- **变式问题**：实验结束后推送 2-3 道相关高考题。

#### 3.3.4 技术要点
- 实验台使用 Canvas/SVG 渲染器材与试剂。
- 状态机记录当前实验阶段、已放置器材、已发生反应。
- 预设实验脚本定义每一步的判定条件与反馈文本。

### 3.4 学习管理工具集成

#### 3.4.1 核心工具
| 工具 | 功能 |
|------|------|
| 课程表 | 周视图课程安排，支持导入/手动编辑，课前提醒 |
| 待办清单 | 任务优先级（高/中/低）、截止日期、与子任务 |
| 番茄钟 | 25/5、45/10、自定义时长，统计每日专注时长 |
| 笔记 | 富文本笔记，支持关联知识点、题目、实验 |
| 倒计时 | 考试倒计时、自定义事件倒计时 |
| 计算器 | 科学计算器，支持常用生物计算（如种群密度、基因频率） |
| 翻译工具 | 中英生物学术语翻译，支持单词本 |

#### 3.4.2 智能学习规划
- **考前规划**：输入考试日期与复习科目，AI 根据错题分布、知识点掌握度生成每日复习计划。
- **任务推荐**：基于当前学习数据，推荐"今天最该做的 3 件事"。
- **学习节奏看板**：整合今日待办、专注时长、复习任务、考试倒计时。

## 4. 技术栈建议

### 4.1 推荐方案
- **前端**：保持现有 HTML5 + Vanilla JS + 自定义组件，逐步引入轻量库：
  - Canvas 动画与实验室：原生 Canvas 2D + 自定义状态机。
  - 富文本笔记：使用开源编辑器如 Quill.js 或 Tiptap（若bundle允许）。
  - 番茄钟/倒计时：Web Worker + Notification API。
- **后端/数据库**：继续使用 Supabase（PostgreSQL + Auth + Storage）。
- **AI 能力**：NVIDIA API（当前已接入）用于 OCR、知识点识别、错误原因分析、动画/实验脚本生成。
- **文件存储**：Supabase Storage 存储错题图片、实验截图、笔记附件。

### 4.2 兼容性
- 支持现代浏览器 Chrome/Edge/Firefox/Safari 最新两个大版本。
- 移动端响应式适配，Canvas 实验支持触屏手势。
- 离线缓存：错题本、笔记、课程表使用 Service Worker + IndexedDB 缓存核心数据。

## 5. 统一设计原则

- **入口统一**：四大模块全部纳入主导航，与现有"练习""社区""复习""悬赏"并列。
- **信用（CR）联动**：
  - 完成错题分析、实验、每日番茄钟奖励 CR。
  - 虚拟实验室操作错误过多适当提示，不扣 CR。
- **知识图谱打通**：错题、动画、实验室均关联同一知识图谱节点，实现"题-图-动画-实验"四维联动。
- **AI 介入有度**：AI 提供识别、推荐、引导，关键决策（如答案判定、实验结束）保留用户确认。

## 6. 优先级与路线图

### 阶段一（MVP，2-3 周）
- 智能错题管理：手动录入 + AI 分析 + 关联练习推送。
- 学习管理工具：待办清单、番茄钟、课程表、倒计时。

### 阶段二（1-2 周）
- Canvas 生物过程可视化：有丝分裂、减数分裂、DNA 复制。
- 虚拟生物实验室：显微镜观察、色素提取。

### 阶段三（1-2 周）
- 错题拍照 OCR、笔记、计算器、翻译工具。
- 虚拟实验室扩展：DNA 粗提取、微生物培养。
- 考前智能规划与"今日最该做"推荐。

## 7. 非功能需求

- **性能**：Canvas 动画在普通笔记本上 60fps；Supabase 查询 8 秒内返回。
- **安全**：错题图片、笔记内容仅用户本人可见；AI 分析结果需用户确认后生效。
- **可访问性**：关键操作支持键盘；图片提供 alt 文本；颜色对比度符合 WCAG 2.1 AA。
- **数据隐私**：不上传含个人身份信息的图片；OCR 结果本地预处理后再发 API。

## 8. 成功指标

- 错题本周活跃率 > 40%。
- 高考/竞赛模式练习完成率提升 15%。
- 虚拟实验室单次平均完成时间 < 8 分钟。
- 学习管理工具每日使用率 > 25%。
